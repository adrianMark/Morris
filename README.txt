Willkommen in unserem Mühle Spiel!

1) Setup
Durch einen einfachen Doppelklick auf Muehle.jar kann die Anwendung gestartet werden. Alternativ kann die Konsole/Terminal genutzt werden und 
durch das eingeben von 'java -jar Muehle.jar' im entsprechenden Verzeichnis wird das Spiel aufgerufen. Auf Windows Maschinen kann die .exe Datei gestartet werden, um 
eine Verknüpfung auf dem Desktop zu erstellen. Es ist wichtig, dass alle Bild Dateien im selben Ordner wie die Muehle.jar Datei verbleiben, da diese von hier in das 
Programm geladen werden.

2) Sonstiges
Der Ordner Code enthält alle Java Source File Dateien und Klassen der Anwendung. Außerdem ist dort das JavaDoc im HTML Format vorzufinden. Der Code ist zusätzlich 
kommentiert.
Der Order enthält eine PDF Dokumentation zu den Grundfunktionen und Klassen der Anwendung.
